Clone online repository(already created repository or project)

1st create a folder using cmd or anything

create folder---> mkdir projects

navigate to that projects ---> cd projects

git init
git clone <- git url link -->

create a local repository and link to online repository

create folder---> mkdir projects

navigate to that projects ---> cd projects

create folder---> mkdir portfolio

create any files like index.html, style.css, etc

initialize the repository(we are creating a memory for storing the local repository or origin)

git init

Now add all new files or modified files to staged area(memory or cotainer)

files already created and modified --> unstaged to staged

files newly created --> untracked to tracked and unstaged to staged

git add .

git restore --staged ..." to unstage

git restore --staged
git restore --staged .

Now commit(it is record for changes ) it is description for change to easy understand and every commit has a unique hash code

uncommit to commit

git commit -m "new files add"

shortcut for add and commit

git commit -am "new files add"

now we are successfully created a updated local repository

now link to online repository and add updates to online repository

git remote add origin <-- link -->
git push origin main

Here origin means location of online repository and main it is a branch(original code project)

shortcut --> if we update the files again 1st commit the local repository and the push command

git push -u origin main

1st time run above code and next every time run below code. it stores origin main in upstream

git push

Update Remote Repo code to Local Repo code

git pull

Rollback Code from Remote Repo to Local Repo

Use git fetch origin followed by git reset --hard origin/main to rollback to the remote state.

Rollback from Local Repo to Working Directory

Use git checkout -- . to discard changes in the working directory.

Create and Merge Branches

Use git branch new-branch to create a new branch.

Use git merge new-branch to merge the new branch into the current branch.
